============
Installation
============


Stable release
--------------

To install power-consumption-logger, run this command in your terminal:

.. code-block:: console

    $ pip install power_consumption_logger

This is the preferred method to install power-consumption-logger, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for power-consumption-logger can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/volker-kempert/power_consumption_logger

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/volker-kempert/power_consumption_logger/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/volker-kempert/power_consumption_logger
.. _tarball: https://github.com/volker-kempert/power_consumption_logger/tarball/master
